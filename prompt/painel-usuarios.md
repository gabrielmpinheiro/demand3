# Implementação do módulo Usuários - Demand3

- Com base nos padrões já estabelecidos no sistema e em nossa API, implemente o módulo usuários, fazendo as chamadas na API e criando as telas que faltam:
  - É preciso que a lista de usuários seja puxada da API, e que a API seja capaz de retornar os dados dos usuários.
  - Ao clicar em Novo Usuário, deve abrir um modal para cadastro de novos usuários.
  - Ao clicar em Editar, deve abrir um modal para edição de usuários.
  - Ao clicar em Excluir, o sistema pergunta se a pessoa tem certeza que deseja excluir o usuário, caso sim, o usuário é excluído.
  - As ações de cadastro, edição e exclusão devem ser feitas através da API.
  - Preciso que seja gerado log para eventos de erro nas ações de cadastro, edição e exclusão, e também para registrar casos de sucesso, salve em /demand3/logs/personal-logs/usuarios.log
  - Verificar a responsividade das novas funcionalidades, se tanto no celular como no desktop estão funcionando corretamente.
